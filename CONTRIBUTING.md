# Contributing to Atlas Platform

Thank you for your interest in contributing to Atlas Platform.

Atlas Platform is a Platform Engineering initiative focused on modern Software Engineering, Cloud Infrastructure, Infrastructure as Code and Developer Experience.

## Mission

Our goal is to build a public reference architecture that demonstrates enterprise-grade engineering practices using:

- Microsoft Azure
- AWS
- Terraform
- Docker
- GitHub Actions
- Cloud Native Technologies

---

## Engineering Principles

All contributions should follow:

- Simplicity
- Scalability
- Maintainability
- Security
- Automation First
- Documentation First

---

## Contribution Workflow

### 1. Fork the repository

Create your own fork.

### 2. Create a feature branch

```bash
git checkout -b feature/my-feature
```

### 3. Implement changes

Ensure code quality and documentation are included.

### 4. Commit changes

```bash
git commit -m "feat: add new feature"
```

### 5. Push branch

```bash
git push origin feature/my-feature
```

### 6. Open Pull Request

Submit a Pull Request against the main branch.

---

## Pull Request Requirements

All Pull Requests should:

- Follow project architecture
- Include documentation updates
- Pass validation checks
- Maintain code readability
- Avoid unnecessary complexity
