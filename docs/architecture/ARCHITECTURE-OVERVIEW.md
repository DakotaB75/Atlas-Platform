# Atlas Platform Architecture Overview

## Mission

Atlas Platform demonstrates how Software Engineering and Platform Engineering can operate together through modern cloud-native practices.

## Core Components

### Application Layer

- NestJS
- TypeScript
- REST APIs

### Data Layer

- PostgreSQL
- Redis

### Platform Layer

- Azure Container Apps
- Azure Container Registry
- Azure Monitor

### Infrastructure Layer

- Terraform
- GitHub Actions

## Design Principles

- Infrastructure as Code
- Automation First
- Observability
- Reusability
- Scalability
- Developer Experience
