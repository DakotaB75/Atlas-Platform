# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for Atlas Platform.

ADRs document important technical decisions and the rationale behind them.

- ADR-001: Terraform as Infrastructure as Code Standard
- ADR-002: Azure Container Apps as Compute Platform
- ADR-003: PostgreSQL as Primary Database
- ADR-004: Redis as Distributed Cache
- ADR-005: GitHub Actions for CI/CD
