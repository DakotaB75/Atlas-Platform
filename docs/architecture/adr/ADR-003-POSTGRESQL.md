# ADR-003

## Title

PostgreSQL as Primary Database

## Status

Accepted

## Context

The platform requires a relational database capable of supporting transactional workloads.

## Decision

PostgreSQL has been selected as the primary persistence layer.

## Rationale

- Open source
- Strong ecosystem
- Cloud-native support
- Reliability
- Industry adoption

## Consequences

Structured application data should be stored in PostgreSQL.
